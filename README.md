## 📖 Overview

**HopePlates** is a powerful social impact platform designed to bridge the gap between resource surplus and those in need. By actively reducing food waste and efficiently redistributing vital resources, our mission is to foster community support and environmental sustainability. 

Our platform seamlessly connects potential donors—ranging from restaurants and event organizers to everyday households—with verified NGOs and dedicated volunteers. This network facilitates the transparent and efficient distribution of surplus food, educational materials (books), and clothing to underprivileged communities.

Additionally, HopePlates empowers users to make secure monetary donations directly to verified NGOs using UPI, maximizing the social impact.

---

## ✨ Key Features

- 🍲 **Multi-Category Donations:** Effortlessly list surplus food, books, and clothes for pickup by registered NGOs or volunteers.
- 📍 **Real-Time Matching:** Employs location-based matching algorithms to instantly connect donors with nearby NGOs and volunteers.
- 🏠 **Home-Based Contributions:** Allows individuals to securely donate home-cooked meals or surplus pantry items to those living on the streets.
- 🛡️ **Stringent NGO Verification:** Features a comprehensive admin dashboard to verify NGO profiles, ensuring complete legitimacy and preventing misuse.
- 🚚 **Pickup Scheduling & Real-Time Tracking:** Empowers volunteers to schedule efficient pickups while providing real-time tracking for transparent distribution.
- 💸 **Seamless Monetary Donations:** Facilitates secure, direct financial contributions to verified NGOs via integrated UPI payments.

## 🛠️ Tech Stack

HopePlates is built using modern, scalable, and responsive web technologies:

**Frontend Ecosystem**
- **Framework:** React.js / Next.js (Optimized for performance and SEO)
- **Styling:** Tailwind CSS / Custom CSS

**Backend Ecosystem**
- **Environment:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT) & Google OAuth
- **APIs:** Google Maps API (Geolocation and Routing)

**Deployment**
- **Frontend Hosting:** Render / Vercel
- **Backend Hosting:** Render

## 🚀 Getting Started

To run HopePlates locally, follow these steps:

### Prerequisites
- Node.js (v16 or higher)
- MongoDB Database (Local or MongoDB Atlas)

### 1. Backend Setup

Navigate to the backend directory, install dependencies, and start the development server:
```bash
cd backend
npm install
# Ensure you create a .env file with the required environment variables first
npm run dev
```

### 2. Frontend Setup

In a new terminal, navigate to the frontend directory:
```bash
cd frontend
npm install
# Ensure you create a .env file with the required frontend environment variables
npm run dev
```
